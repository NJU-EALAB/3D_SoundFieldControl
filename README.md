# 三维声场控制

## 主要功能

基于听众位置设计二维扬声器阵列的FIR滤波器，使用二维扬声器阵列控制其三维声辐射特性，同时实现垂直方向与水平方向的声场控制.

# ThreeDim-Sound-Field-Control

Design FIR filters based on the audience positions for generating homogenous sound field over the audience.
